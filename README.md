# Nacho
A lightweight matrix lib in C++
## Get start
To define a matrix instance with code:
```cpp
CMatrix mat = CMatrix(rowNum, colNum);
```
To set the element at 2nd row and 3rd column to be 1.5:
```cpp
mat.set(2, 3, 1.5);
```
To get the element at 2nd row and 3rd column:
```cpp
mat.get(2, 3);
```
